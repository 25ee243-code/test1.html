Git Operations Assignment

1. Aim

To perform basic Git operations such as initializing a repository, adding and committing files, creating and switching branches, and merging branches.

2. Description

Git is a distributed version control system used to track changes in source code during software development. It helps developers manage different versions of files and collaborate efficiently.

The following operations are performed:

  • Initialize a Git repository
  • Add and commit files
  • Create and switch branches
  • Merge branches

3. Algorithm

  1. Create a project directory
  2. Initialize Git using git init
  3. Create a file and add content
  4. Add file using git add
  5. Commit file using git commit
  6. Create a branch using git branch
  7. Switch branch using git checkout
  8. Modify file and commit changes
  9. Switch back to main branch
  10. Merge branch using git merge

4. Execution and Testing

Initialize Repository

mkdir my-project
cd my-project
git init

Add and Commit Files

echo "Hello Git" > file.txt
git add file.txt
git commit -m "Initial commit"

Create and Switch Branch

git branch feature-branch
git checkout feature-branch

Modify and Commit in Branch

echo "New feature added" >> file.txt
git add file.txt
git commit -m "Added new feature"

Merge Branches

git checkout main
git merge feature-branch

Delete Branch (Optional)

git branch -d feature-branch

Result

The Git operations were executed successfully. The repository was initialized, files were committed, branches were created and switched, and merging was completed successfully.
